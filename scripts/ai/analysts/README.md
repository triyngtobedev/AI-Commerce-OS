# AI Analyst

## Objetivo

Responsável por analisar produtos encontrados pelo sistema e determinar seu potencial comercial utilizando Inteligência Artificial.

---

## Entrada

O módulo recebe produtos no formato:

- Nome do produto.
- Categoria.
- Visualizações.
- Curtidas.
- Comentários.
- Outros dados coletados.

---

## Processo

A IA deve analisar:

- Potencial de viralização.
- Interesse do público.
- Nível de concorrência.
- Possibilidade de venda.
- Qualidade da oportunidade.

---

## Saída

O módulo retorna:

```json
{
  "potencial": "",
  "score": 0,
  "motivo": ""
}
