# AI-Commerce-OS

> Sistema de automação para pesquisa de produtos, análise com IA e geração de conteúdo para e-commerce.

## Objetivo

Criar uma plataforma capaz de:

- Encontrar produtos virais.
- Analisar o potencial de venda usando IA.
- Gerar roteiros UGC.
- Criar prompts para geração de vídeos.
- Automatizar todo o fluxo utilizando n8n.

## Tecnologias

- Docker
- n8n
- Google Gemini
- Git
- GitHub
- VS Code
